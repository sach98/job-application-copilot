# Mac always-on host setup — replaces Oracle VM

candidate's Mac is the 24/7 host. n8n + scrapers + Playwright + CLIs all run locally. Cloudflare Tunnel exposes webhooks + the Tinder review app to the internet.

## Pre-existing on candidate's Mac (verified)

- node v25.9.0
- python 3.11.1
- Homebrew 5.1.12
- claude 2.1.150 (Claude Code CLI, Pro plan auth)
- codex (Homebrew install)
- agy (Antigravity CLI)

## Installed during setup

- **cloudflared** — Cloudflare Tunnel daemon (via brew)
- **n8n** — workflow runner (via npm global)
- **python-jobspy** — multi-source job scraper (via pip --user)
- **playwright + chromium** — form-fill bot runtime (via pip --user + chromium download)

## Manual setup (sudo required — candidate runs)

### 1. Disable Mac sleep on AC power

```bash
sudo pmset -c sleep 0       # don't sleep on AC
sudo pmset -c disksleep 0   # don't sleep disks on AC
sudo pmset -c displaysleep 5 # display only — keeps screen off, system awake
sudo pmset -c womp 1        # wake on network for SSH/automation
pmset -g                    # verify
```

### 2. Launchd plist for n8n auto-start

`~/Library/LaunchAgents/com.example.jobhunt.n8n.plist` — auto-start n8n at boot + on login + restart on crash.

(Content generated by Codex during build step; manual install via `launchctl bootstrap gui/$UID ~/Library/LaunchAgents/com.example.jobhunt.n8n.plist`.)

### 3. Cloudflare Tunnel

Free Cloudflare account, one-time tunnel auth:

```bash
cloudflared tunnel login                    # opens browser → pick domain
cloudflared tunnel create jobhunt           # create named tunnel
cloudflared tunnel route dns jobhunt n8n.<your-domain>     # subdomain → tunnel
cloudflared tunnel route dns jobhunt review.<your-domain>  # for Tinder app
```

Then `~/.cloudflared/config.yml`:

```yaml
tunnel: jobhunt
credentials-file: ${HOME}/.cloudflared/<tunnel-uuid>.json
ingress:
  - hostname: n8n.<your-domain>
    service: http://localhost:5678
  - hostname: review.<your-domain>
    service: http://localhost:5678
  - service: http_status:404
```

Start: `cloudflared tunnel run jobhunt`. Persist via launchd plist too.

### 4. n8n first-run

```bash
N8N_HOST=n8n.<your-domain> \
N8N_PROTOCOL=https \
N8N_PORT=5678 \
WEBHOOK_URL=https://n8n.<your-domain>/ \
GENERIC_TIMEZONE=Asia/Kolkata \
N8N_BASIC_AUTH_ACTIVE=true \
N8N_BASIC_AUTH_USER=reviewer \
N8N_BASIC_AUTH_PASSWORD='<choose strong password>' \
n8n start
```

Verify: `curl -u reviewer:<password> https://n8n.<your-domain>/healthz` → `{"status":"ok"}`.

## Risks vs Oracle VM

| Risk | Mitigation |
|------|-----------|
| Mac sleeps / closes lid | `pmset -c sleep 0` + keep lid open OR run with lid closed using external display unplugged trick (`sudo pmset -c lidwake 0`) |
| Power outage | Battery (laptop) bridges short cuts. UPS for desktops. |
| Reboot loses tunnels | launchd auto-restarts both n8n + cloudflared on boot |
| Mac being used by candidate actively | Workload is mostly idle except scrape windows (09:00 / 12:30 / 16:00) + brief Sonnet calls. CPU under 5% most of day. |
| Local IP/router change | Cloudflare Tunnel handles NAT — works behind any home router, no port-forward needed |
| Privacy: Mac is personal device | n8n protected by basic-auth + Cloudflare Access policy (only candidate's Google account) |

## Backups

Nightly 02:00 IST:
- `tar czf ~/JobHunt/backups/n8n-$(date +%F).tgz ~/.n8n` (n8n state)
- `rclone copy ~/JobHunt/backups gdrive:JobHunt/backups` (off-Mac)

Cron entry: `0 2 * * * cd ~ && tar czf ~/JobHunt/backups/n8n-$(date +\%F).tgz ~/.n8n && rclone copy ~/JobHunt/backups gdrive:JobHunt/backups --max-age 14d`.
